docbook rocks!

it just takes a bit for it to be normal

how to make a paragraph

  <para>
    your text goes here! It isn't whitespace sensitive.
  </para>

  <para>
    Feel free to break up your sentences in to

    as many lines
    as
    you
    like.
  </para>

how about a bit of code

  <literal>printf()</literal>

and it works in titles and paras and what-not

  <title>Printing With <literal>printf()</literal></title>

  <para>
    to print some stuff to the screen, use <literal>printf()</literal>.
  </para>

how to show console output

  <screen>
    $ git clone https://github.com/grahamc/docbook.rocks.git
  </screen>

but if you want to make it even better,

  <screen>
    <prompt>$ <prompt>git clone https://github.com/grahamc/docbook.rocks.git
  </screen>
and then you can add some display code to prevent the $ from being copied: 
  .prompt {
    user-select: none;
    -moz-user-select: none;
  }
